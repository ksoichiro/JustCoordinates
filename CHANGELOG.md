# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2026-02-20

### Added

- J key toggle to show/hide coordinates HUD
- Minecraft 1.16.5, 1.17.1, 1.18.2 support for Fabric and Forge
- Quilt compatibility for 1.21.10-1.21.11

### Changed

- Restructured subproject directories from flat to hierarchical layout

## [0.1.1] - 2026-02-12

### Added

- Minecraft 1.21.7-1.21.11 support
  - Quilt compatibility for 1.21.7-1.21.9 (1.21.10+ requires Fabric Loader 0.17.3+ which Quilt does not yet support)

## [0.1.0] - 2026-02-11

### Added

- XYZ coordinates HUD display in the top-left corner with gray background and white text (Bedrock Edition style)
- Format: `Position: X, Y, Z`
- Auto-hide when F3 debug screen is displayed
- Auto-hide when HUD is hidden (F1 key)
- Multi-loader support: Fabric, NeoForge, Forge
- Minecraft version support: 1.19.2, 1.20.1, 1.21.1, 1.21.3-1.21.11
